# 软件问题

:::info 教程使用 Reivt2019 版

- 安装Reviet2019时,会下载Revit 内容库,安装时间受网速影响。 
- 可以断网安装，待完成后重新修复 Revit Content Libraries 2019 ，安装内容库(样板文件及族库)
- 高版本无法降级至低版本, 如Revit2019无法降级为2018。
- 首次打开低版本软件需要升级；

:::
