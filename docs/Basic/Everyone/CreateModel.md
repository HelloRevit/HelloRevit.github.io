# 创建项目

模型是建筑设计的虚拟版本。它描述了模型元素的几何形状，并捕获了元素之间的设计意图和逻辑关系。
## 创建模型
准备好开始新设计时，请创建 Revit 模型。在模型的一个视图中对图元所做的更改会在所有其他视图中立即可见。
