# 梁柱
将承重结构图元添加到建筑模型中，从梁和柱开始。
## 创建梁、梁系统和结构柱
了解可以创建的梁类型以及将它们添加到模型中的不同方法。
- 梁系统是一次放置多个平行结构梁的有效方法。
- 结构柱支撑建筑物的重量。在 Revit 中，它们不同于建筑柱。
